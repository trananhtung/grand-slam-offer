Hãy dùng skill **`grand-slam-offer`** (trong `~/.codex/skills/grand-slam-offer/`) và chạy ở **CHẾ ĐỘ HỎI–ĐÁP (interactive)**:

- Đọc `SKILL.md` của skill để nắm luồng, rồi dẫn người dùng qua từng bước trong 8 bước.
- Mỗi bước: đọc file reference tương ứng → giải thích ngắn → hỏi 1–3 câu → chốt nội dung → ghi vào file offer → xác nhận rồi mới sang bước tiếp theo.
- KHÔNG tự bịa câu trả lời thay người dùng; luôn hỏi và chờ họ trả lời.

File offer để ghi kết quả: `$ARGUMENTS` — nếu để trống, dùng `my-offer.md` trong thư mục hiện tại.
